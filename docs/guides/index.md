# Guías Prácticas

Guías paso a paso para usar y trabajar con el proyecto.

## Guías Disponibles

- [Workflow con curl](./workflow-curl.md) - Guía completa para usar la API con curl, incluyendo verificación del sistema, gestión de la cola y manejo de errores
- [Generación de Textos Meteorológicos](./prompt-generacion-textos-meteorologicos.md) - Guía para generar textos meteorológicos en español que sean parseables automáticamente, incluyendo formato, campos obligatorios/opcionales, ejemplos y prompt para agentes de IA
